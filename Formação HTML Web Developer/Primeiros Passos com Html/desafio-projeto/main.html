Sobre as Tags ol e ul

( Voltar )

São listas que na verdade são listas ordenadas e não ordenadas.

Lista Ordenada -> ol , segue uma lista que contem uma ordem. Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Lista não ordenada -> ul , segue uma lista que não contem uma ordem especifica. Exemplo:

  • Item 1
  • Item 2
  • Item 3

Sobre a Tag sub

( Voltar )

Essa tag define basicamente um texto subscrito, faz um texto ficar menor e ficar na parte de baixo. Exemplo:

O Famoso H2O


Sobre a Tag sup

( Voltar )

Ao contrario da tag sub A tag sup deixa o texto menor , porém ela deixa na parte do topo. Exemplo:

Este texto contém texto sobrescrito

Sobre a Tag abbr

( Voltar )
